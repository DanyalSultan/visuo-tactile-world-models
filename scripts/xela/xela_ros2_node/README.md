# ROS2 node for Xela 

## Overview

We largely use the Xela ROS node provided by the company, but we have made minor changes and recommend using this fork: 

```
https://github.com/akashsharma02/xela_server_ros2
```

Once the ros2 package is installed, on a terminal you should be able to run: 

```
ros2 run xela_server_ros2 xela_server_ros2
```
to start the xela publisher
